03 / CONSULTATION

같은 기능도,
이렇게 설명합니다.

기술 사양부터 나열하지 않습니다. 고객님의 서비스에 왜 필요한지, 무엇을 준비하고 결정하면 되는지부터 이해하기 쉽게 안내합니다.

Case 01
필라테스
고객님
“작은 필라테스 스튜디오 홈페이지를 만들고 싶은데, 어떤 기능이 필요한지 전혀 모르겠어요.”
일반적인 기술 안내

5페이지 반응형 웹으로 제작되며 상담 폼, 카카오맵 API, SEO 메타태그, SSL 인증서와 호스팅 세팅이 포함됩니다.

제가 드리는 안내

먼저 방문자가 궁금해할 프로그램, 강사 소개, 이용 가격과 위치부터 보여주면 좋겠습니다. 고객 대부분이 휴대폰으로 보기 때문에 글과 사진을 작은 화면에 맞게 정리하고, 전화나 카카오톡 상담 버튼을 쉽게 찾을 수 있게 배치하겠습니다.

대표 사진과 프로그램 내용만 보내주시면 홈페이지에 맞는 순서로 정리해 드립니다. 사이트 주소 구매와 공개 과정도 제가 진행합니다.

우선 문의는 전화, 카카오톡, 문의 양식 중 어떤 방식으로 받는 것이 가장 편하신가요?

Case 02
병원
고객님
“진료과와 의료진을 소개하고 온라인 예약도 받고 싶은데, 어떤 방식으로 만들어야 할까요?”
일반적인 기술 안내

반응형 웹으로 제작되며 의료진·진료과 CMS, 예약 폼, 지도 API, SEO, SSL 인증서와 관리자 페이지를 구성할 수 있습니다.

제가 드리는 안내

환자는 주로 어떤 진료를 받을 수 있는지, 누가 진료하는지, 오늘 방문할 수 있는지, 어떻게 예약하는지를 먼저 확인합니다. 첫 화면에서 진료과와 진료시간을 빠르게 찾고 바로 전화하거나 예약할 수 있도록 구성하겠습니다.

온라인 예약은 날짜를 남기면 직원이 연락하는 간단한 방식과, 병원 일정에 연결해 가능한 시간을 바로 고르는 방식이 있습니다. 현재 운영 방식에 맞춰 관리하기 편한 쪽을 안내해 드립니다.

현재 예약은 전화, 네이버 예약, 병원 프로그램 중 어떤 방식으로 관리하고 계신가요?

Case 03
모바일 초대장
고객님
“기존 템플릿 말고 원하는 스타일로 커스텀하고 싶은데, 디자인을 어떻게 설명해야 할지 모르겠어요.”
일반적인 기술 안내

커스텀 UI, 반응형 원페이지, 스크롤 애니메이션, 갤러리 슬라이더, 지도 API와 참석 여부 폼을 적용할 수 있습니다.

제가 드리는 안내

디자인 용어로 설명하지 않으셔도 괜찮습니다. 마음에 드는 초대장이나 사진처럼 원하는 분위기와 가까운 이미지 2~3개만 보내주세요. 공통된 느낌을 찾아 색상, 글씨체, 사진 배치와 화면의 움직임으로 정리해 드립니다.

예식 정보, 사진, 지도와 연락처는 하객이 휴대폰에서 쉽게 찾도록 구성합니다. 완성된 템플릿에 내용만 바꾸는 것이 아니라 두 분의 사진과 분위기에 맞춰 함께 조율합니다.

초대장을 처음 본 분들이 따뜻하다, 세련됐다, 특별하다 중 어떤 느낌을 받았으면 좋으신가요?
